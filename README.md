## Linux Projects: Pharmacy Management System

In these projects, I developed a Pharmacy Management System using two different approaches in Linux.

# The first project was implemented using Bash scripting (Shell), where I created a simple system to manage medicines, handle basic operations like adding, searching, and updating records through command-line interaction.

# The second project was implemented using Python, where I built a more advanced version of the system with better structure and extended functionality. It includes improved data handling, user interaction, and additional features for managing pharmacy operations more efficiently.

These projects demonstrate the difference between scripting and programming approaches in Linux, and how the same system can be developed with different levels of complexity and flexibility.
